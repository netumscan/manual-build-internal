传输Code ID字符#

Code ID 字符用于标识已扫描条形码的代码类型。这在解码多种代码类型时非常有用。Code ID 字符插入到前缀字符（如果已选择）和解码的符号之间。

Code ID 标识符#

AIM Code 标识符#

每个 AIM 代码标识符包含三个字符串 ]cm，其中：

  • ] = 标志字符

  • c = 代码字符（见表）

  • m=修饰符
